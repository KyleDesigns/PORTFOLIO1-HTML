UI Courses Taking Career Foundry’s UX courses gave me a better beginner’s understanding of what UX consists of. I was given a lot of resources, reading assingments, quizzes, and tests. Here is where I really honed in on everything UX/UI. I was also able to learn the basics of coding HTML, CSS, and JavaScript. Besides being taught more about design along with homework, we also had projects. Working individually and in groups, I worked on 5 projects and building case studies for each. One project we got to interact with a real client from a non-profit organization for a website redesign for desktop, tablet, and mobile. With these courses I was able to learn more intermediate level understanding with UX Design and learn more about the UI side of things. Aside from reading materials and tests, I was able to get hands on with design software through projects. Having 1-on-1 with a Professional Mentor was extremely valuable!
